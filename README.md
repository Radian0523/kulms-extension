# KULMS Extension

京都大学LMS ([KULMS](https://lms.gakusei.kyoto-u.ac.jp/)) のUIをカスタマイズするChrome拡張機能です。

## 機能一覧

### 1. テーマ切り替え
画面右下のテーマパネルから、4種類のテーマに切り替えられます。
- **デフォルト** — 標準の白背景
- **ダーク** — 暗い背景で目に優しい
- **セピア** — 温かみのある色合い
- **ブルー** — 青系の落ち着いたトーン

設定は `chrome.storage` に保存され、次回アクセス時も維持されます。

### 2. 全科目課題一覧パネル
画面右端にスライドインするパネルで、全科目の課題を一覧表示します。
- Sakai Direct API (`/direct/assignment/`) から各コースの課題を取得
- 期限順にソート
- 期限が近い課題をハイライト表示
- パネル開閉時にページコンテンツを左に押し出して表示（オーバーレイではなく押し出し方式）

### 3. 授業資料ツリービュー
授業資料（リソース）ページで、フォルダ/ファイルの階層構造を視覚的に表示します。
- フォルダとファイルの深さに応じたインデント
- フォルダ行は青い左ボーダー＋薄い背景色＋太字
- ファイル種別ごとの左ボーダー色（PDF=赤、画像=緑、動画=紫、コード=グレーなど）
- フォルダの展開/折りたたみに対応（MutationObserver）

### 4. サイドバー科目名の整理 + ピン留めソート
- 科目名から年度・学期情報を除去して短縮表示（例: `[2026前期月1]パワーエレクトロニクス` → `[月1]パワーエレクトロニクス`）
- ピン留めした科目を曜日・時限順に自動ソート（月1 → 月2 → 火1 → ... → 金5）
- 時間割に該当しない科目はソート末尾に配置

### 5. サイドバー: コース行クリックで展開/折りたたみ
- 科目名のテキスト部分だけでなく、行全体をクリックでツールリストの展開/折りたたみが可能
- シェブロンボタンやピンボタンは従来通りの動作を維持

### 6. サイドバー: ツール表示管理
各科目のツールリストを整理し、よく使うツールだけを表示します。
- **デフォルト表示**: 概要、授業資料（リソース）、課題
- それ以外のツールは「その他 ▶」トグルの下に折りたたみ
- 科目ごとにツールの表示/非表示をカスタマイズ可能（ホバーで表示される +/− ボタン）
- 設定は `localStorage` に保存

### 7. サイドバーリサイズ
- サイドバーの右端にドラッグハンドルを表示
- ドラッグでサイドバーの幅を自由に変更可能（120px〜500px）
- サイドバー内のテキストがリサイズに追従して折り返し位置を更新
- 幅は `localStorage` に保存され、次回アクセス時も維持

## インストール方法

1. このリポジトリをクローン
   ```bash
   git clone https://github.com/Radian0523/kulms-extension.git
   ```
2. Chrome で `chrome://extensions` を開く
3. 右上の「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. クローンしたフォルダを選択

## ファイル構成

```
kulms-extension/
├── manifest.json     # Chrome拡張マニフェスト (Manifest V3)
├── content.js        # メインのコンテンツスクリプト（全機能のIIFE）
├── styles.css        # テーマ・各機能のスタイル
├── background.js     # サービスワーカー（開発用）
└── hot-reload.sh     # ファイル変更監視スクリプト（開発用）
```

## 対応環境

- Google Chrome (Manifest V3)
- 対象サイト: `https://lms.gakusei.kyoto-u.ac.jp/*` (Sakai LMS)
