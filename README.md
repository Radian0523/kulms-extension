# KULMS+

**[ランディングページ](https://radian0523.github.io/kulms-extension/)** | **[Chrome Web Store](https://chromewebstore.google.com/detail/kulms+/akfadmompgbhncnocomalofhcihpejjb)**

京都大学LMS ([KULMS](https://lms.gakusei.kyoto-u.ac.jp/)) のUIをカスタマイズするブラウザ拡張機能です。Chrome / Edge / Firefox に対応しています。

## 機能一覧

ヘッダー右端の「☰」ボタンからパネルを開き、**課題・教科書・Tips・設定**の4タブで操作できます。全機能を搭載し、設定タブから個別にオン/オフできます。

### 1. ツールバーポップアップ
ブラウザのツールバーにある拡張機能アイコンをクリックすると、課題一覧をポップアップで表示します。

**アクセス方法（Chrome / Edge）:**
1. ツールバー右上の拡張機能ボタン（パズルピースのアイコン 🧩）をクリック
2. 一覧から「KULMS+」を見つけてクリック → ポップアップが表示される
3. ピン留めアイコン（📌）をクリックすると、ツールバーにアイコンが常時表示される
4. 以降はツールバーの KULMS+ アイコンをワンクリックで開ける

- LMS ページを開かなくても、キャッシュされた課題データを緊急度別に確認可能
- 更新ボタンで LMS タブのコンテンツスクリプト経由で最新データを取得
- 課題クリックで LMS の課題ページを新タブで開く
- ダークモード対応（`prefers-color-scheme`）

### 2. 全科目課題一覧パネル
パネルの「課題」タブで全科目の課題・クイズを一覧表示します。
- Sakai Direct API (`/direct/assignment/`, `/direct/sam_pub/`) から各コースの課題・クイズを取得
- 緊急度別セクション（遅延提出 / 緊急 / 注意 / 余裕 / その他 / 完了済み / 削除済み）
- 課題カードにコース名ピルバッジ、残り時間（日・時間・分）、チェックボックス付き
- チェックした課題は「完了済み」セクションに移動（永続保存）
- 削除した課題は「削除済み」セクションに移動し、30日後に自動で完全削除
- 削除済みアイテムは「元に戻す」ボタンで復元可能
- 課題名をクリックするとポータル内で該当科目の課題ページに遷移
- クイズ/テストには「テスト」バッジを表示

### 3. メモ機能
課題タブの下部にある「＋」ボタンからメモを追加できます。
- テキスト入力に加え、コース選択（任意）と締切日時（任意）を設定可能
- モバイル（タッチデバイス）ではモーダルオーバーレイで入力（PC はインラインフォーム）
- モーダル内の締切日時は select ドロップダウン式カスタムピッカーで入力（Android WebView 互換）
- 締切付きメモは通常の課題と同様に緊急度セクションに統合表示
- 締切なしメモは専用の「メモ」セクションに表示
- メモもチェック・削除が可能（削除は確認付き、削除済みセクションに移動）
- `chrome.storage` に永続保存

### 4. 教科書パネル
パネルの「教科書」タブでKULASISシラバスから教科書・参考書情報を取得して表示します。
- 一度取得したデータはキャッシュされ、次回以降は即表示（手動更新も可能）
- 各科目のシラバスページへの直リンク付き
- Amazonへのリンク付き

### 5. 緊急度カスタマイズ
設定タブから緊急度の閾値と色をカスタマイズできます。
- **閾値設定**: 緊急（時間）、注意（日）、余裕（日）をそれぞれ数値で指定
- **カラー設定**: 緊急・注意・余裕・その他の4色をカラーピッカーで変更
- セクションヘッダー、ピルバッジ、サイドバー色分けに反映

### 6. 多言語対応 (i18n)
- ブラウザの言語設定に応じてUIを自動で日本語/英語に切り替え
- 設定タブから言語を手動で上書き可能（自動 / 日本語 / English）

### 7. ファイル ドラッグ＆ドロップ
課題提出ページにドロップゾーンを追加し、ファイルをドラッグ＆ドロップで添付できます。
- `#addSubmissionForm` 内の添付パネルにドロップゾーンを挿入
- ドラッグ中はハイライト表示、ドロップ後はファイル名をピル表示
- ドロップゾーンクリックでファイル選択ダイアログを開くことも可能
- `DataTransfer` API で `<input type="file">` にファイルをセットし `change` イベントを発火

### 8. 授業資料ツリービュー
授業資料ページで、フォルダの展開/折りたたみをページ遷移なしで実行できます。
- 「ツリービュー拡張」をONにすると、Finder/VSCode 風のツリー表示に変更
- collectionId パスベースの深さ判定によるインデント付き階層表示
- フォルダ行に展開/折りたたみ矢印 (▸/▾)、太字表示
- 行ホバーハイライト、キーボードナビゲーション（↑↓←→ Enter）
- コンパクトレイアウト（行高さ削減、アクションボタンはホバー時のみ表示）
- ファイルアイコン非表示、ダークモード対応

### 8a. TA 採点支援（TAロール専用）
Sakai の個別採点画面（Grader UI）と採点一覧画面に、採点作業を効率化する機能を追加します。
- **未採点ジャンプ**: 「次/前の未採点へ」ボタンで提出済み・未採点の学生へワンクリックで移動
- **状態アイコン**: 受講者選択ドロップダウンと採点一覧テーブルに、提出/採点状態を示すアイコンを表示
- **残数バッジ**: ヘッダーに「未採点提出 N 件」をリアルタイム表示
- **Offcanvas 操作改善**: 採点パネルを開いたまま背景の PDF 等を操作可能に
- **未保存ガード**: 採点内容を変更した状態でページを離れようとすると確認ダイアログを表示
- **凡例ツールチップ**: 各アイコンの意味を確認可能
- page-world bridge (`grading-ta-page.js`) で `<sakai-grader>` の `originalSubmissions` を安全に取得
- 学生ロールでは Sakai サーバー側でガードされるため影響なし

### 8b. Tips タブ
使い方のヒントやアップデート情報を表示するタブです。
- GitHub Pages から Tips データ（JSON）を取得してカード形式で表示
- デフォルトでは日替わりランダム2件を表示（日付シードによる決定的シャッフル）
- 更新ボタンで別のランダム2件にシャッフル
- 「すべて表示」ボタンで全件表示、「今日のTipsだけ表示」で2件に切り替え
- 1時間キャッシュ（`chrome.storage`）

### 9. 科目名の整理 + ピン留めソート
- 科目名から年度・学期情報を除去して短縮表示（例: `[2026前期月1]パワーエレクトロニクス` → `[月1]パワーエレクトロニクス`）
- ピン留めした科目を曜日・時限順に自動ソート（月1 → 月2 → 火1 → ... → 金5）

### 8.5. 授業中ハイライト
- 京大の時間割（1限 8:45〜10:15 ... 5限 16:45〜18:15）に基づき、現在の時限に該当する科目のサイドバーに赤い「NOW」バッジを表示
- 休み時間中は次の時限の科目に青い「NEXT」バッジを表示（1限は開始15分前から、5限後は非表示）
- 60秒ごと＋タブ復帰時に自動更新
- `[曜日+時限]` パターンを持つ科目名のみが対象（デフォルトOFF）

### 9. コース行クリックで展開/折りたたみ
- 科目名のテキスト部分だけでなく、行全体をクリックでツールリストの展開/折りたたみが可能

### 10. ツール表示管理
各科目のツールリストを整理し、よく使うツールだけを表示します。
- **デフォルト表示**: 概要、授業資料（リソース）、課題
- それ以外のツールは「その他 ▶」トグルの下に折りたたみ
- 科目ごとにカスタマイズ可能（ホバーで表示される +/− ボタン）

### 11. サイドバーリサイズ
- サイドバーの右端をドラッグで幅を変更可能（120px〜500px）
- 幅は `localStorage` に保存され、次回アクセス時も維持

### 12. 科目タブ色分け
- サイドバーの各科目に、最も近い締切の緊急度に応じて色分け表示
- 3つの表示モード（設定から切り替え）:
  - **ボーダー**: 左端に4pxのカラーライン（デフォルト）
  - **背景**: 行全体を薄い色で塗り + 3pxボーダー
  - **太バー + 背景**: 6pxの太ボーダー + 軽い背景色
- 色はカスタマイズ可能（デフォルト: 赤/黄/緑/灰）
- 背景色は `color-mix()` で自動計算されるため、カスタムカラーにも追従
- ページ読み込み時にキャッシュから自動適用（パネルを開かなくても表示）

### 13. 新着課題バッジ
- 新しい課題が追加されたコースにサイドバー上で赤丸バッジを表示

### 13.5. ピン留め上部バー
- ヘッダー直下にピン留め済み科目を横並びのバーで表示（PC のみ）
- スクロールしても追従する常時表示（`position: fixed`）
- サイズは 4 段階（小 / 中 / 大 / 特大）から選択可能
- 科目タブ色分け（ボーダー / 背景 / 太バー + 背景）に完全対応
- NOW / NEXT / 新着課題バッジをサイドバーからクローン表示
- 「コース行クリック展開」ON 時は、タブをクリックするとツール一覧（概要・授業資料・課題等）がドロップダウンで展開（「ツール表示管理」設定も継承）
- KULMS+ パネルの押し出しモード時は上部バーも一緒にスライド

### 14. サイドバースタイル変更
- 選択中の科目のデフォルト青背景を除去し、左ボーダーによる選択表示に変更
- 科目名のフォントサイズ・太さを非選択時と統一

### 15. パネル押し出し表示
- パネルをページに重ねて表示（デフォルト）か、ページを横に押して表示かを選択可能

### 16. 個別設定
- 設定タブからグループ化されたセクション（外観・パネル・サイドバー・コースページ・課題更新・緊急度・開発者）で機能を管理
- デフォルトON: 教科書パネル、科目タブ色分け
- 設定は `chrome.storage` に永続保存

## インストール方法

### Chrome Web Store（推奨）
[Chrome Web Store](https://chromewebstore.google.com/detail/kulms+/akfadmompgbhncnocomalofhcihpejjb) からインストールできます（Edge でも利用可能）。

### Chrome / Edge（手動）
1. このリポジトリをクローン
   ```bash
   git clone https://github.com/Radian0523/kulms-extension.git
   ```
2. `chrome://extensions`（Edge: `edge://extensions`）を開く
3. 「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. クローンしたフォルダを選択

### Firefox
1. `about:debugging#/runtime/this-firefox` を開く
2. 「一時的なアドオンを読み込む」をクリック
3. クローンしたフォルダの `manifest.json` を選択

> **Note**: Firefox でストア配布する場合は `./build.sh firefox` で生成した ZIP を使用してください。

## ビルド

`build.sh` でブラウザ別の配布用 ZIP を生成できます（要 `jq`）。

```bash
./build.sh chrome   # Chrome / Edge 用 ZIP
./build.sh firefox  # Firefox 用 ZIP（manifest に gecko 設定を追加）
./build.sh all      # 両方生成（デフォルト）
```

## 対応環境

- Google Chrome / Microsoft Edge / Mozilla Firefox (Manifest V3)
- 対象サイト: `https://lms.gakusei.kyoto-u.ac.jp/*` (Sakai LMS)
