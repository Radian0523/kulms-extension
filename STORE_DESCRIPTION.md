京都大学LMS（KULMS）のUIを大幅に強化するブラウザ拡張機能です。Chrome / Edge / Firefox に対応。
ヘッダー右端の「☰」ボタンからパネルを開き、課題・教科書・設定の3タブで操作できます。全機能を設定タブから個別にオン/オフできます。

【ツールバーポップアップ】
1. ツールバー右上の拡張機能ボタン（🧩）をクリック → 一覧から「KULMS+」を選択
2. ピン留め（📌）すればツールバーにアイコンが常時表示され、ワンクリックで開けます
LMSページを開かなくても、キャッシュされた課題を緊急度別にサッと確認できます。更新ボタンでLMSタブから最新データを取得。ダークモード対応。

【課題一覧パネル】
全科目の課題・クイズを一画面で確認。緊急度別（遅延提出 / 緊急 / 注意 / 余裕 / その他）に自動分類し、残り時間を日・時間・分で表示します。課題の完了チェック、削除（復元可能）、提出状態の自動検出に対応。定期的に自動更新されます。

【メモ機能】
自分用のメモを追加できます。コース選択や締切日時を設定すると、通常の課題と同様に緊急度セクションに統合表示されます。

【教科書パネル】
KULASISシラバスから各科目の教科書・参考書情報を自動取得。シラバスページへの直リンクとAmazonへのリンク付きで、教科書探しが楽になります。

【緊急度カスタマイズ】
緊急・注意・余裕の閾値（時間/日）と色を自由に変更できます。

【多言語対応】
日本語 / 英語の自動切り替え。設定から手動で言語を切り替えることもできます。

【サイドバー強化】
- 科目タブ色分け: 締切の緊急度に応じて色分け（ボーダー / 背景 / 太バー + 背景 の3モード）
- 新着課題バッジ: 新しい課題が追加されたコースに赤丸バッジ
- 科目名の整理: 年度・学期情報を除去して短縮表示
- ピン留めソート: ピン留め科目を曜日・時限順に自動ソート
- 授業中ハイライト: 現在の授業に「NOW」、休み時間は次の授業に「NEXT」バッジを表示
- コース行クリック: 行全体クリックでツール展開/折りたたみ
- ツール表示管理: よく使うツールだけを表示し、それ以外は「その他」に折りたたみ。各ツールにホバーで表示される +/- ボタンで、科目ごとに表示/非表示をカスタマイズ可能
- サイドバーリサイズ: ドラッグで幅を変更可能
- スタイル変更: 選択中科目の表示をすっきりとした左ボーダーに変更

【ピン留め上部バー】
- ヘッダー直下にピン留め科目を横並びのバーで常時表示
- サイズ 4 段階（小 / 中 / 大 / 特大）、色分けスタイル（ボーダー / 背景 / 太バー + 背景）に対応
- NOW / NEXT / 新着課題バッジもサイドバーと連動
- コース行クリック展開 ON 時は、タブクリックでツール一覧（概要・授業資料・課題等）がドロップダウン表示

【授業資料ツリービュー】
授業資料ページのフォルダをページ遷移なしで展開/折りたたみ。

【その他】
- パネル押し出し表示: パネルをページに重ねる or ページを横に押す、を選択可能
- プレビューモード: ダミー課題でUIを確認できる開発用モード
