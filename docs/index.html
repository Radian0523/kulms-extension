ダウンロード

Chrome / Edge Firefox準備中 Safari準備中 iOS準備中 Android準備中

主な機能

📋

課題一覧

全科目の課題を締め切り順にまとめて表示。提出状況もひと目で確認。

📚

教科書パネル

教科書・参考書情報をサイドバーからすばやくアクセス。

📝

メモ

科目ごとにメモを残せる。課題のヒントや授業ノートに。

🎨

サイドバー色分け

科目ごとにサイドバーを色分け。視認性がアップ。

🔧

ツール整理

使わないツールを非表示にしてサイドバーをすっきり。

✂️

科目名整理

長い科目名を自動で短縮。ピン留め科目を上部に固定。
